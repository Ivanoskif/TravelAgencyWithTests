## Overview
This project is a web application for a travel agency, developed using ASP.NET Core MVC.  
The main focus of the application is the integration of external API services, as well as functional testing and quality assurance (QA).

## Features
- Travel offers and destination management
- Integration with external API services
- Structured MVC architecture
- Emphasis on functional testing and application quality
- Clean and maintainable codebase

## Technologies Used
- ASP.NET Core MVC
- C#
- SQLite
- Playwright
- xUnit
- FluentAssertions
- NSubstitute

## Project Purpose
This project was developed as an individual learning project with a strong focus on:
- Backend web development with ASP.NET Core MVC
- External API integration
- Automated functional testing and QA practices

### Prerequisites
- .NET SDK (version compatible with the project)
- SQLite
